# NextJS List

## Overview

Simple site made with Next.js.

## Tech Stack

NextJS

## Run Locally

- Clone the project

```bash
git clone https://github.com/theanujdev/next-list-site.git
```

- Go to the project directory

```bash
cd next-list-site
```

- Install dependencies

```bash
npm install
```

- Start the server

```bash
npm run dev
```
